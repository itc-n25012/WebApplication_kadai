# 基本情報技術者試験対策 Web アプリ

## 概要

基本情報技術者試験（午前問題）の学習を目的とした Web アプリケーション。

## 使用技術

- Next.js (App Router)
- TypeScript
- ESLint
- CSS Modules
- microCMS
- GitHub
- Vercel

## 機能一覧

- 問題一覧表示
- 問題詳細表示
- microCMS による問題管理
- 分野別学習
- レスポンシブ対応
- 学習履歴機能
- 問題復習機能

## ページ構成

- トップページ：問題一覧表示
- 問題詳細ページ：問題文・選択肢・解説表示

## デプロイ

Vercel を使用して公開
